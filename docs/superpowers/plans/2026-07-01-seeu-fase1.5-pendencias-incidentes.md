# SEEU Fase 1.5 — Pendências de Incidentes — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Raspar também a aba "Pendências de Incidentes" do SEEU (fluxo próprio: dropdown de Juízo), roteando qualquer item para a triagem — defensivo, para nunca perder um incidente pendente (raro).

**Architecture:** Estende `_async_scrape_mesa` com um caminho especial para a aba `pendencias` (itera o `<select>` de Juízo em vez de SITUACOES) + uma função pura `parse_pendencias` (defensiva). As outras abas ficam intactas. A jusante reusa 100% o pipeline da Fase 1 (staging → 5_TRIAGEM, dedup processo+Seq/content_hash).

**Tech Stack:** Python 3.12 (`enrichment-engine/.venv/bin/python`) + pytest; browser CDP (patchright) no daemon do defensor.

## Global Constraints

- Só a aba **Pendências de Incidentes**; o loop das outras abas (SITUACOES) fica intacto.
- **Defensivo:** `parse_pendencias` tenta `Seq→CNJ`; se não casar mas houver conteúdo, captura 1 item cru (não perde); tabela vazia → `[]`.
- Item vai para `seeu_import_staging` com `ato="Pendência de incidente — verificar"`, `conteudo="Mesa do Defensor\n"+bloco` (sentinela SEEU), rota `5_TRIAGEM`.
- Read-only sobre o SEEU (nunca clica ação). `parse_pendencias` nunca lança.
- Descoberta genérica do `<select>` de Juízo (`querySelectorAll('select')`), sem id fixo. Sem `<select>` → trata aba como vazia.
- Testes: pytest loader **exec-strip-main** (padrão dos outros `test_*.py` do worker).

# Tabela sem itens (caso usual das Pendências de Incidentes).
_TABELA_VAZIA_RE = re.compile(r"nenhum registro|não há|nao ha|sem pend[êe]ncias", re.IGNORECASE)


def parse_pendencias(texto: str) -> list:
    """Parser DEFENSIVO da aba Pendências de Incidentes (layout distinto e raro).
    Tenta o padrão Seq→CNJ das outras abas; se não casar mas houver conteúdo real,
    devolve UM item cru (seq=None, cnj=None) para não perder o incidente; tabela
    vazia → []. Nunca lança.
    Retorna list[tuple[int|None, str|None, str]] = (seq, cnj, bloco)."""
    t = (texto or "").strip()
    if not t or _TABELA_VAZIA_RE.search(t):
        return []
    try:
        blocos = _split_blocos_por_processo(t)
    except Exception:
        blocos = []
    if blocos:
        return blocos
    # Conteúdo fora do padrão conhecido → captura defensiva (1 item cru).
    return [(None, None, t)]
```

# Pendências de Incidentes: aba com <select> de Juízo (não SITUACOES). Descoberta
# genérica (sem id fixo): pega os selects visíveis do frame e devolve as options.
JS_LIST_JUIZOS = r"""() => {
  const sels = Array.from(document.querySelectorAll('select'));
  // Heurística: o select de juízo tem várias options com texto (>1). Pega o maior.
  let best = null, bestN = 1;
  for (const s of sels) {
    const opts = Array.from(s.options || []).filter(o => (o.textContent || '').trim());
    if (opts.length > bestN) { best = s; bestN = opts.length; }
  }
  if (!best) return [];
  return Array.from(best.options).map(o => ({ value: o.value, label: (o.textContent||'').trim() }))
                                 .filter(o => o.label);
}"""

JS_SELECT_JUIZO = r"""(value) => {
  const sels = Array.from(document.querySelectorAll('select'));
  let best = null, bestN = 1;
  for (const s of sels) {
    const opts = Array.from(s.options || []).filter(o => (o.textContent || '').trim());
    if (opts.length > bestN) { best = s; bestN = opts.length; }
  }
  if (!best) return false;
  best.value = value;
  best.dispatchEvent(new Event('change', { bubbles: true }));
  if (typeof best.onchange === 'function') best.onchange();
  return true;
}"""
```
